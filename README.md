# 🏪 Pre-Entrega de Proyecto - E-Commerce

> Una herramienta de interfaz de línea de comandos (CLI) diseñada para interactuar de forma asíncrona con servicios de catálogo de productos.

---

## 🛠️ Stack Tecnológico

| Tecnología | Responsabilidad en el Sistema |
| :--- | :--- |
| **Node.js** | Entorno de ejecución asíncrono para la capa de servicios. |
| **JavaScript (ES6+)** | Lógica de programación, manipulación de estructuras de datos y promesas. |
| **Fetch API** | Comunicación HTTP nativa con servicios REST externos. |

---

## 📐 Diseño Arquitectónico y Principios de Software

Este desarrollo fue concebido bajo estándares de arquitectura limpia, priorizando la separación de responsabilidades.
